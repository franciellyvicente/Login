# 🔐 Página de Login

Interface de página de login desenvolvida com HTML5 e CSS3, com foco na prática de estruturação de formulários e estilização de interfaces web. Projeto criado para aprimorar conhecimentos em layout, organização de elementos e design de formulários responsivos.

🔗 **Demo:** [franciellyvicente.github.io/Login](https://franciellyvicente.github.io/Login/)

## 🛠️ Tecnologias

- HTML5
- CSS3

## 📋 Funcionalidades

- Campo para usuário ou e-mail
- Campo para senha
- Opção "Lembrar senha"
- Link "Esqueci minha senha"
- Link para cadastro
- Botão de login

Layout desenvolvido para ser simples, organizado e intuitivo, simulando a interface de autenticação de aplicações web.

## 🎨 Design

- Paleta de cores em preto, bege e vermelho
- Ilustração de fundo na página
- Tipografia: Poppins

## 📌 Observações

- Este projeto possui apenas interface visual
- Não possui integração com back-end ou sistema de autenticação
- Desenvolvido com fins educacionais e de prática em front-end
